spinet

Spinet it is a vintage musical instrument which has similar looks of a keyboard. It always get better when Technology touches the vintage stuffs. So we decided to build a simple Electronic Spinet using Arduino without using any keys. This project use of ultrasonic sensor which replace the keys.The complete working of the spinet is based on the distance parameter. If there is an obstacle in front of the ultrasonic sensor, sound will be produced in the speaker. Here obstacle is nothing but our finger which we are using for producing the sound.
